Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript (ES6) – Application logic and dynamic behavior
Local Storage API – Data persistence across browser sessions

User enters an expense title and amount.
On clicking Add Expense, the data is:
Stored in an array
Saved to the browser's Local Storage
Expenses are dynamically rendered on the page.
Users can:
Edit an expense
Delete an expense
The total expense amount updates automatically.
Data remains saved even after refreshing or reopening the browser.
